HTML and CSS

XHTML and CSS are codes that makes websites work.

Editors

websites at their basic level are text files. All you need to write a website is a simple text editor.

Tags

XHTML language revolves around tags. Everything in a website is contained within a tag. The first tag you make on a website tells its doctype the kind of document your file wil be.

scope

most tags have opening and closing tags. everything in a web object is in the opening and closing tags. everything outside the tahs belong to another object.

head

the head of a site have information for the browser. the information in it isnt seen by the user. it has keywords, titles, browser information etc.

body

the body of the site has all the site content like paragraphs, images, audio, animations, backgrounds and more.

comments

you can add to website documents like python, java, or other programming language.

header

a header is a system of section titles arranged from h1 to h6 in decenting order.

tool tips

tool tips let users hover over parts of a page for further explination.

adding images to your site

to adjust the size of an image you need to adjust the height attribute.
